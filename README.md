# Telegram File Share Bot

A Telegram bot built with **Node.js** and the `node-telegram-bot-api` library, designed to share files through automatically generated links. Ideal for keeping files organized and secure in a storage group without exposing it directly to end users.

---

- **üîó Permanent Links**: Each file uploaded to the storage group receives a unique and permanent link.
- **üåê Universal Access**: Links can be shared with anyone, even if they are not in the storage group.
- **üõ°Ô∏è Enhanced Privacy**: Your original group (where you store the files) will not interact with the user, so if any file has copyright issues, the bot will bear the consequences, not the entire group with all your files (as this feature can encourage piracy, please use the bot wisely).
- **üìÇ Automatic Organization**: Files are managed automatically, without manual intervention.

## üöÄ How It Works

1. **Create a Telegram group** (preferably private) to serve as the file storage.
2. **Add the bot as an administrator** to the group, granting read and write permissions.
3. **Send a file to the group** and the bot will automatically generate a unique link.
4. **Share the link** with anyone; when clicked, the bot will send the corresponding file to the user.

## ‚öôÔ∏è Setup

### 1. Clone the Repository
```bash
git clone https://github.com/antoinepdev/telegram-file-share-bot.git
cd telegram-file-share-bot
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure Environment Variables
Create a `.env` file in the project root with the following variables:

| Variable | Description | How to Obtain |
|----------|-------------|---------------|
| `BOT_TOKEN` | Bot authentication token | Generated by [@BotFather](https://t.me/BotFather) |
| `STORAGE_GROUP_ID` | ID of the group where files will be stored | Obtain by adding [@RawDataBot](https://t.me/RawDataBot) to the group and copying the `"chat"."id"` value |
| `BOT_URL` | Public bot link (e.g., `https://t.me/YourBotName`) | Provided by @BotFather when creating the bot |

#### Example `.env` file:
```env
BOT_TOKEN=717jfjfngncn:AAFpl5chxhdvwk4n1yKpznj91tmqZ3chfjfr
STORAGE_GROUP_ID=-10024748473658
BOT_URL=https://t.me/birdsender_bot
```

### 4. Start the Bot
```bash
npm run prod
```
If configured correctly, you will see the message `"works..."` in the console.
